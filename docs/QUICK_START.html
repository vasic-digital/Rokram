Yole - Quick Start Guide

Get started with Yole in 5 minutes across all platforms.

Platform Setup

Android

./gradlew :androidApp:assembleDebug
adb install androidApp/build/outputs/apk/debug/androidApp-debug.apk

Desktop

./gradlew :desktopApp:run

Web

./gradlew :webApp:wasmJsBrowserRun

iOS

Open iosApp/iosApp.xcodeproj in Xcode and build.

Supported Formats

Yole supports 18+ markup formats including Markdown, todo.txt, CSV, LaTeX, and more.

Development Commands

# Build all
./gradlew clean build

# Run tests
./gradlew test
