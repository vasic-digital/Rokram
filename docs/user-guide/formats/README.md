# Format Guides Index

Complete guides for all 17 formats supported by Yole.

---

## ‚úÖ Complete Guides (17/17)

### Core Formats
1. **[Markdown](./markdown.md)** (.md, .markdown, .mdown, .mkd)
   - CommonMark + GitHub Flavored Markdown
   - Headers, links, lists, tables, code blocks, task lists

2. **[Plain Text](./plaintext.md)** (.txt, .text, .log)
   - Universal format, syntax highlighting for code
   - Perfect for notes, logs, snippets

3. **[Todo.txt](./todotxt.md)** (.txt)
   - Task management format
   - Priorities, projects, contexts, due dates

4. **[CSV](./csv.md)** (.csv)
   - Comma-separated values
   - Tables, data, spreadsheet compatibility

### Technical Formats
5. **[LaTeX](./latex.md)** (.tex, .latex)
   - Academic and scientific documents
   - Mathematical equations, citations

6. **[Org Mode](./orgmode.md)** (.org)
   - All-in-one organizational system
   - TODO items, scheduling, tables, code blocks

---

### Wiki Formats
7. **[WikiText](./wikitext.md)** (.wiki, .wikitext, .mediawiki)
   - MediaWiki markup language
   - Wikipedia-compatible syntax
   - Templates, categories, magic words

8. **[Creole](./creole.md)** (.creole, .wiki)
   - Standardized wiki markup
   - Cross-wiki compatibility
   - Minimal, consistent syntax

9. **[TiddlyWiki](./tiddlywiki.md)** (.tid, .tiddler)
   - Non-linear personal wiki
   - Tiddler format (metadata + content)
   - Tags and transclusion

### Technical Formats
10. **[AsciiDoc](./asciidoc.md)** (.adoc, .asciidoc, .asc)
    - Technical documentation format
    - Powerful features for complex docs
    - Converts to HTML, PDF, EPUB

11. **[reStructuredText](./restructuredtext.md)** (.rst, .rest)
    - Python documentation standard
    - Sphinx integration
    - Directives and roles

### Specialized Formats
12. **[Key-Value](./keyvalue.md)** (.properties, .ini, .env, .conf)
    - Configuration file formats
    - Java properties, INI files
    - Environment variables

13. **[TaskPaper](./taskpaper.md)** (.taskpaper, .todo)
    - Plain-text task management
    - Projects, tasks, and tags
    - Flexible organization

14. **[Textile](./textile.md)** (.textile, .txtl)
    - Lightweight markup language
    - Clean, readable syntax
    - Used in Textpattern CMS

### Data Science
15. **[Jupyter](./jupyter.md)** (.ipynb)
    - Interactive notebook format
    - JSON-based cell structure
    - Code, markdown, and outputs

16. **[R Markdown](./rmarkdown.md)** (.Rmd, .rmarkdown)
    - Reproducible R documents
    - Markdown + R code chunks
    - Multiple output formats

### Other
17. **[Binary](./binary.md)** (All binary files)
    - Binary file detection
    - Prevents editing non-text files
    - Safe file handling
